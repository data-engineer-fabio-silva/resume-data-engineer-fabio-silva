# Fabio Silva – Data Engineer Resume

Static HTML/CSS resume site for Fabio Silva, focused on Senior Data Engineer roles.

## How to view

- Open `index.html` directly in your browser, or
- Serve the folder with any static server (for example, a simple Python server):

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000/index.html` in your browser.

## Tech stack

- HTML5 + CSS3
- Google Fonts (Raleway)
- Font Awesome icons
